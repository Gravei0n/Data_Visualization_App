# Advanced Data Visualization App

This is an advanced data visualization application built using Streamlit. The app is designed to help users visualize various types of business data effectively. It supports multiple visualization types, allowing users to gain insights from their data quickly and easily.

## Features

- **Bar Chart:** Compare expenses across different departments.
- **Line Chart:** Show revenue and profit trends over time.
- **Pie Chart:** Display the distribution of expenses across departments.
- **Scatter Plot:** Analyze the relationship between revenue and profit.
- **Heatmap:** Visualize the correlation between different numerical values.
- **Box Plot:** Show the distribution of expenses within departments.
- **Gantt Chart:** Present task timelines with start and end dates.
- **Map Visualization:** Display geographical data with latitude and longitude.
- **Word Cloud:** Visualize keyword frequencies in the text data.
- **Bubble Chart:** A variation of a scatter plot where the size of the bubbles represents an additional dimension of the data.
- **Area Chart:** Similar to a line chart, but the area below the line is filled in.
- **Tree Map:** Used to display hierarchical data as a set of nested rectangles.
- **Violin Plot:** Combines aspects of box plots and kernel density plots to show data distributions.
- **Donut Chart:** A variation of the pie chart with a hole in the center.
- **Waterfall Chart:** Used to show cumulative effects of sequentially introduced positive or negative values.
- **Sankey Diagram:** Used to visualize flows and their quantities in a system.
- **Radar Chart:** Displays multivariate data in the form of a two-dimensional chart.
- **Sunburst Chart:** A radial equivalent of a tree map.
- **Bullet Chart:** Used for comparing a metric against a target.
- **Bubble Map:** Combines bubble charts and map visualizations to show the geographical distribution of data points.
- **Network Diagram:** Displays relationships between different entities in a network.
- **Chord Diagram:** Visualizes the inter-relationships between data in a matrix format.
- **Timeline:** Shows a sequence of events in chronological order.


## How to Use

1. **Upload Data:**
   - Upload a CSV or Excel file containing your data. The app expects data with specific columns to generate the visualizations correctly.

2. **Select Visualization:**
   - Choose from various chart types using the dropdown menu in the sidebar. Each chart type will use relevant columns from the dataset.

3. **Generate Visualizations:**
   - The app will generate and display the selected visualization based on the uploaded data. You can view detailed information about each chart type in the sidebar.
